## Клиенты приложения

### Веб‑клиент (MVC)

- Реализован в проекте `Tp_CourseWork` через контроллер `HomeController`.
- Основные возможности:
  - просмотр списка локаций и фильтрация по мэру;
  - просмотр деталей по выбранной локации;
  - создание, редактирование и удаление локаций (CRUD);
  - просмотр статистики, рассчитанной через облачную функцию Yandex.Cloud.
- Для работы веб‑клиента необходимо запустить проект `Tp_CourseWork` (профиль HTTPS).

### GUI‑клиент (WPF)

- Реализован в отдельном проекте `LocalityGuiClient` (WPF под .NET 6).
- Основные возможности:
  - указание базового URL REST‑сервиса (по умолчанию `https://localhost:49105`);
  - загрузка списка локаций через эндпойнт `GET /api/GetLocalities`;
  - отображение списка в таблице и детальной информации по выбранной записи.
- Требования:
  - запущен сервер `Tp_CourseWork` (REST‑сервис);
  - установлен .NET 6.0 SDK/Runtime с поддержкой WPF.
